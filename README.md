# FoodWagen - Full Starter Project

This is a full starter scaffold for the FoodWagen assessment built with Next.js (App Router) + TypeScript + Tailwind.

## Quickstart

1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

3. Run tests:
   npm run test

Notes:
- API base: https://6852821e0594059b23cdd834.mockapi.io/Food
- Custom CSS classes must start with `food-` per assessment guidelines.
